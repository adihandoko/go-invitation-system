# Frontend React

Frontend React terpisah untuk `go-invitation-system`.

## Menjalankan project

1. Salin env:

```bash
cp .env.example .env
```

2. Install dependency:

```bash
npm install
```

3. Jalankan development server:

```bash
npm run dev
```

Default frontend akan memakai API di `http://localhost:8080`.

## Fitur

- Melihat daftar invitation
- Membuat invitation baru
- Melihat detail invitation dari slug
- Mengirim RSVP
- Melihat statistik RSVP dan daftar tamu
